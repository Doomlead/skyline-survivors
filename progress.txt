## What We Should Work On Next

The highest-leverage next milestone is **Challenge Mode Productization**.

### Why this should be next
- It closes one of the two largest remaining design-document compliance gaps.
- It reuses existing combat systems with limited net-new content risk.
- It creates a durable replayability pillar and clear telemetry for future balancing.

### One-Task Update (Next Sprint)

**Task:** Implement **Challenge Mode v1 (Last Stand + Assault Only)**.

**Definition of Done:**
1. Add mode-select entry points for Last Stand and Assault Only.
2. Persist high scores / run summaries per mode.
3. Show post-run report screen (time survived / waves / score / district outcomes).
4. Gate mode-specific modifiers cleanly from core campaign state.
5. Add automated tests for mode initialization, score persistence, and report payload formatting.
